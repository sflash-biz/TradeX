Overview
This interface will show you hits not connected with regular trades. Interface more simple because we not need to return traffic to this type of trades or setup\count forces.. Some system trades have no "in" stats because we can't count it by virtue of the features.

    no-cookie - Clicks without cookies.
    no-image - Clicks for browsers which don't load images. (Work only with images.php tool: Global Settings - "Filter image.php")
    no-referrer - Raw in and clicks for surfers with empty HTTP_REFERER
    non-trade - Raw in and clicks for surfers with referrer which doesn't match any NON system trader.
    not-counted - Clicks exciting max clicks amount (Global Settings - "Count Clicks") from an individual surfer, (additional clicks) will be tallied as this system trade.
    search-engine - Raw in and clicks for surfers which referred from search engines list.
    spiders - Raw in and clicks for browsers\hosts with useragents matched spiders list or EMTY useragent data (strlen(useragent) <= 25).
    unknown - Raw in and clicks for browsers\hosts with bad in and click signature (Cookie, File or Cache data key).

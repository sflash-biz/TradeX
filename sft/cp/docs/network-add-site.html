Network - Add a Site
Table of Contents
Overview
From this dialog you will be able to add network sites. The network sites feature allows you to link several SFTrade installations together so you can view the stats from all of your sites and sync settings between them.
Network Site Settings
When adding or editing a network site there are several settings you can configure. Those are as follows:
    Control Panel URL - Enter the base URL of the SFTrade control panel, without the index.php filename. For example: http://www.networksite.com/sft/cp/
    Username - Enter the username for the SFTrade control panel
    Password - Enter the password for the SFTrade control panel
    Owner - Enter the site owner's name (useful when selecting sites to sync)
    Category - Enter the category of the site (useful when selecting sites to sync)
    Display stats - Check this box to view stats from this site in the Network > Stats interface

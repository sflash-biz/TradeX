Moving Your SFTrade Installation
Table of Contents
Overview
Use these instructions when it is necessary to move your SFTrade site to a new server, or to a new account on the same server. These instructions only account for moving your SFTrade installation and not any other parts of your site.
Moving Steps
Moving your installation to a new server or a new account on the same server can be accomplished using the following steps:
  1. Copy the SFTrade files from the old server to the new server. The directory where SFTrade is installed on the new server should match the directory name on the old server.

  2. Set the permissions on all files in the data directory to 666 (this includes all files in all sub-folders)

  3. Set the permissions on all folders in the data directory to 777 (this includes all folders in all sub-folders)

  4. Login to the SFTrade control panel, access the Global Settings interface and save your settings.
Once these have been done, you should be set to go.

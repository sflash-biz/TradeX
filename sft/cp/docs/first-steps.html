First Steps After Installation
Table of Contents
Recommended Steps
  1. If you prefer to set your own control panel username and password, use the Settings > Change Login function in the control panel menu.

  2. If you will be allowing new trades to register, setup your trade rules which will appear on the registration form. Use the Settings > Trade Rules function in the control panel menu to configure the trade rules that you would like to have displayed.

  3. To start tracking the surfers coming into your site you will need to setup the incoming stats tracking script code on one or more pages of your site. That is detailed in the Stats Tracking > Incoming section of this software manual.

  4. To start tracking the surfers going out from your site and to start sending surfers to your trades, you will need to setup the outgoing stats tracking script. That is detailed in the Stats Tracking > Outgoing section of this software manual.

  5. If you want to be able to assign your sites to categories, use the Settings > Categories function in the control panel menu to configure the categories you would like to use.

  6. If you want to be able to assign your sites to groups, use the Settings > Groups function in the control panel menu to configure the group names you would like to use.
